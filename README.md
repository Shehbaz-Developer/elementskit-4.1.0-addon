# Elementskit Addon

**Elementor Addon Plugin** with advanced widgets, social feeds, WooCommerce enhancements, and visual effects.

**Developer:** Shehbaz Developer

---

## Features

- Advanced Tab, Accordion, Toggle, Slider, and Unfold widgets  
- Social media feeds: Facebook, Instagram, Twitter, Dribbble, Pinterest, Yelp  
- WooCommerce enhancements: Product carousel, Mini cart, Category list  
- Visual effects: Liquid glass, Glass morphism, Image hover effects, Parallax effects, Fancy animated text  
- Advanced controls for responsive design, icons, colors, typography, and transitions  
- Protected content widget, Conditional widget support  
- Optimized for performance and compatibility with latest Elementor versions  

---

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`  
2. Activate the plugin through the **Plugins** menu in WordPress  
3. Access widgets through Elementor editor
